## REQUIREMENTS

*   Ruby 2.6+
*   Rails 5.2+
*   ActiveRecord 5.2+
*   PostgreSQL 9.5+
*   [PostgreSQL extensions](https://github.com/Casecommons/pg_search/wiki/Installing-PostgreSQL-Extensions) for certain features

## INSTALL

Create database
```
$ rake db:create
```

Run migrations
```
$ rake db:migrate
```

Prepare basic information
```
$ rails db:seed
```
